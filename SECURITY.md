# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Email:** security@agentika.io

**Alternatively:** Open a [GitHub issue](https://github.com/agentika-labs/agentika-plugin-marketplace/issues) if the vulnerability is not sensitive.

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Any potential impact

We will acknowledge your report within 48 hours and aim to provide a fix within 7 days for critical issues.

## Scope

This policy applies to the plugin marketplace repository and its published plugins. Third-party dependencies are outside our direct control but we will coordinate disclosure as needed.
